# Giphy GIF Demo

Animated gif with lazyloading using paging3 library. used local database for favorite feature.


# Features
- Trending GIF files
- Search on trending page
- Favorite any GIF from trending and search result
- View the list of Favorite GIFs

# Library used
- RoomDatabase
- Paging3
- Kotlin
- MVVM
- Retrofit
- Glide
